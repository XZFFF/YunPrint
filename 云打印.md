## 云打印（YunPrint）

注：URL前缀为：http://localhost/YunPrint/public/

线上URL前缀为：http://yunprint.applinzi.com/YunPrint/public/index.php

### 用户（user）

#### 登录接口

> POST user/login/oklogin

| 参数名   | 参数类型 | 备注 |
| -------- | -------- | ---- |
| username |          |      |
| password |          |      |

```json
{
    "errcode": "0",
    "errmsg": "登录成功",
    "data": {
        "id": 1,
        "username": "xzfff",
        "realname": "谢泽丰",
        "tel": "13125177868",
        "time": "2018-03-27 08:45:11"
    }
}
```

| -------- | -------- | ---- |

| realname |          |      |

#### 创建订单

> POST user/order/createorder

| 参数名     | 参数类型 | 备注          |
| ---------- | -------- | ------------- |
| sid        |          |               |
| file1id    | int      | 文件id        |
| file1num   |          | 打印份数      |
| file1color |          | 0-黑白 1-彩色 |
| file1style |          | 0-单面 1-双面 |
| file2id    |          |               |
| file2num   |          |               |
| file2color |          |               |
| file2style |          |               |
| file3id    |          |               |
| file3num   |          |               |
| file3color |          |               |
| file3style |          |               |

```json
{
    "errcode": "0",
    "errmsg": "创建成功",
    "data": 1
}
```
